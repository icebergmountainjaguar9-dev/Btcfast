# Btcfast

A static website for `btcunlimited123.com` that simulates Bitcoin transaction acceleration by transaction hash.

## Files created

- `index.html` — main website content and form
- `styles.css` — responsive site styling
- `script.js` — simulated mempool acceleration flow

## Deployment

This project is ready to deploy as a static site on Vercel. Simply connect the repository and deploy the root project.

## Usage

1. Open `index.html` in a browser.
2. Enter a 64-character Bitcoin transaction hash.
3. Submit the form to see the simulated acceleration response.
